You can use a USB device for storage
by setting up the fstab like this and
plugging in a USB key to store files:

proc            /proc           proc    defaults          0       0
PARTUUID=bb0c0d39-01  /boot           vfat    defaults,ro          0       2
PARTUUID=bb0c0d39-02  /               ext4    defaults,ro,noatime  0       1
tmpfs           /tmp            tmpfs   nosuid,nodev      0       0
tmpfs           /var/log        tmpfs   nosuid,nodev      0       0
tmpfs           /var/tmp        tmpfs   nosuid,nodev      0       0
# a swapfile is not a swap partition, no line here
#   use  dphys-swapfile swap[on|off]  for that
/dev/sda1	/media/usb_key	vfat	rw,user,suid,nodev,auto,uid=1000 0 2

The last line adds a usb_key auto mount at the location /media/usb_key

You need to (as root on the pi) make the usb_key directory, global
permissions and owned by the user pi.
